# VITALIS — Fitness & Wellness Platform

Фитнес-платформа с калькулятором BMI, планами тренировок и прайсингом.

## Стек
- HTML + CSS + Vanilla JS
- Шрифты: Montserrat, Inter
- Тема: Лайм-голубой градиент, тёмные/светлые секции

## Навыки
- Рабочий BMI-калькулятор (SVG ring)
- Pricing cards с toggle месяц/год
- CSS particles в hero
- Градиентные карточки программ
- Counter анимации
- Двуязычность EN/RU
